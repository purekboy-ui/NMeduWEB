# 溫暖又富有人性的核醫 (Warm and Humane Nuclear Medicine)

這是一個專注於核子醫學衛教的網站，旨在以溫暖、富有人性的方式，向大眾介紹核子醫學的檢查與治療項目。透過清晰的資訊與友善的介面，消除民眾對「輻射」的恐懼，並提供正確的醫學知識。

## 網站特色

本網站包含以下主要區塊：

- **關於核醫 (About)**：以輕鬆幽默的口吻介紹核子醫學科，打破冰冷的醫療形象。
- **環境設施 (Facilities)**：展示科內的先進儀器與舒適環境，如 SPECT、PET/CT 等，並搭配輪播功能。
- **常見問題 (FAQ)**：整理民眾最關心的問題（如：檢查會痛嗎？輻射會影響健康嗎？），提供詳盡解答。
- **服務項目 (Services)**：詳細列出各項核醫檢查與治療項目，包含：
    - FDG 正子掃描
    - 骨骼掃描
    - 鐳-223 治療
    - 鎵-67 炎症/腫瘤掃描
    - 心肌灌注掃描
    - 放射性碘-131 治療
    - 腎臟掃描 & 腎臟皮質攝影
    - 甲狀腺掃描、唾液腺掃描、副甲狀腺掃描
    - 腦部多巴神經元斷層造影 & 腦灌注檢查
    - 腎上腺皮質顯像
- **圖庫 (Gallery)**：展示相關影像與活動紀錄。

## 技術架構

- **HTML5**: 語意化標籤結構。
- **CSS3**: 使用 Flexbox 與 Grid 佈局，實現響應式設計 (RWD)，並包含自定義動畫與模態框 (Modal) 樣式。
- **JavaScript (Vanilla JS)**: 
    - 導覽列滾動效果與手機版選單切換。
    - 圖片輪播 (Carousel) 功能。
    - 滾動視差與淡入動畫 (Intersection Observer)。
    - 模態框 (Modal) 動態內容載入。

## 部署與執行

本專案為靜態網頁，可直接於瀏覽器開啟 `index.html` 瀏覽，或部署至 GitHub Pages、Firebase Hosting 等靜態網站託管服務。
